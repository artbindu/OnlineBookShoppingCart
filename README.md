# Online Book Shopping Cart

JSP-based e-commerce application for online book shopping.

## Tech Stack
JSP • Oracle DB • Tomcat 9.0 • HTML/CSS/JS

## Features
**User**: Login/Signup, Search Books, Shopping Cart, Wishlist, Card/Wallet Payment, Order History, Profile Management  
**Admin**: Add Books, ISBN Validation, Inventory Management

## Quick Setup
1. Configure DB in `DBconnection.jsp` (Oracle XE @ localhost:1521)
2. Deploy to Tomcat webapps
3. Access: `http://localhost:12345/onlinebookshoppingcart/OnlineShoppingCart/`

## Database
- ARTBINDU_BOOK_USERINFO
- ARTBINDU_BOOK_PRODUCTINFO  
- ARTBINDU_CARDDETAILS
- ARTBINDU_ORDERDETAILS

See `problems.txt` for known DB connection issues.
